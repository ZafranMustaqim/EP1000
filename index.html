EP1000 Digital Fabrication and Prototyping

Digital Fabrication and Prototyping(DFAB) is a module that teaches students on methods of digital creation. The skills that a student will learn when taking this module is learning how to use the software when 3D Printing is involved. The student will also learn how to use a laser cutter as well as learn to use Arduino Uno which is used to create our final project.

This site was created to document my journey studying this module
