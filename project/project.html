My Project

For my project, I decided to make a door sensor with an alarm.

This door sensor can be used for home security as when the door is opened, an alarm will sound alerting people in the house. It can also be used to protect the privacy of your room.

The sensor revolves around this magnetic switch that sends a signal to Arduino that the door is moving. If the door is opened, the circuit will be open sounding an alarm inside the box that contains the Arduino.
