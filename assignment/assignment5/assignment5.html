Assignment 5

For this assignment, we had to use git to push our website onto GitHub. Git is a distributed version control system designed for tracking changes in any set of files. GitHub is a code hosting platform that allows version control and collaboration.

Steps to push

  • The first step is to create a repository in Github.
  • Next step is to open Git Bash and to create a local project that is directed to the current working directory.
    • Use the command pwd to print out the current directory.
    • Use the command cd 'path name' to move to the specified path in your local computer.
  • Next is to use the command git init to initialise the repository.
  • Use the command git add to add all the files to the given folder.
    • You can use the command git status to view all the files that are going to be staged to the first commit.
  • After uploading the file, use the command git commit -m 'your message' to commit the content with a description of what was changed and why it was changed.
  • Next, copy the URL of your GitHub's remote repository and use the command git remote add origin 'your URL name'.
  • Finally, push the whole code into your GitHub repository by using the command git push -u origin master and input your GitHub username and password.
