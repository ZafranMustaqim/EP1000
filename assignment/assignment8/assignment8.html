Parametric Box

For this assignment, we were tasked to create a box using parametric design. What this means is that we create a function that will store the value of a side of the box. When this value is changed, the size of the box will change accordingly.

Fusion 360 model

This is how the model looks like in Fusion 360. There are 5 parts to make the box. The 4 sides and the bottom of box. The box is held by finger tabs that are laser cut.

The box
